從零開始的網頁設計課程

轉職網頁開發人員　掌握就業必備職能
由淺入深學習技術　強化職場的競爭力

HTML

HTML 是基礎網頁技術，網頁瀏覽器可
以讀取 HTML 檔案，成為視覺化網頁。

CSS

CSS 網頁樣式表，可彈性自定變化網頁
元素顯示樣貌，節省工作時間。

JavaScript

JavaScript 是最流行的腳本語言，能在
靜態頁面上添加一些動態效果。

jQuery

以 JS 為基礎的函式庫，號稱能讓開發
者能寫的少，做的多！

Boostrap

熱門的前端框架工具，用於設計響應式
網站、排版、格式和介面。

Sass

Sass 是一個CSS 預處理器。可以幫助我們
減少CSS 重複的代碼，節省開發時間。

Vue.js

Vue.js 是一個用於建立使用者介面和單
頁應用的 JavaScript 框架。
